Getting Started
What is Virtual Screen Manager?
Virtual Screen Manager gives you up to nine independent virtual workspaces on each physical monitor. Apps stay open and running in the background — switching workspaces is instant and nothing ever closes or restarts.
How is this different from Windows virtual desktops?
Windows virtual desktops require you to manually manage which apps appear on each desktop and don't preserve taskbar visibility for hidden apps. Virtual Screen Manager keeps all your apps running in the taskbar, switches workspaces instantly, and automatically assigns new windows to whichever workspace you're on when you open them.
What are the system requirements?
Windows 10 (build 1903 or later) or Windows 11. Single or multiple physical monitors. 1920×1080 or higher recommended.
Do I need administrator privileges to run Virtual Screen Manager?
No. It runs as a standard user. As a consequence, it cannot manage applications that are themselves running as administrator (Task Manager, Registry Editor, etc.) — this is a Windows security boundary.
Switching Workspaces
How do I switch workspaces?
Three ways: click any cell in the grid panel, move your mouse to a screen edge (edge navigation), or use keyboard hotkeys (Ctrl+Alt+A through Ctrl+Alt+I by default).
My hotkeys conflict with another application. Can I change them?
Yes. Open Settings and change the Hotkey Modifier. Ctrl+Shift and Alt+Shift are available alternatives. The workspace letters (A–I) are fixed.
Can I turn off edge navigation?
Yes. Toggle "Mouse edge switching" in Settings, or press Ctrl+Alt+M at any time. The state is saved automatically.
Edge navigation keeps firing when I don't want it to. What do I do?
Increase the Edge Hold Time slider in Settings. Most users find 400–700ms comfortable. You can also turn edge navigation off entirely if you prefer to use only clicks and hotkeys.
Managing Applications
Will my applications close when I switch workspaces?
Never. Virtual Screen Manager moves apps off-screen and brings them back. They stay running in exactly the state you left them.
An application isn't showing up in my workspace. Why?
Two common causes: it's running with elevated privileges (e.g. Task Manager) and can't be tracked — this is a Windows limitation; or it was open before Virtual Screen Manager started. Switch away and back — discovery runs continuously and will pick it up within a few seconds.
I can't find an application — it seems to have disappeared.
It's almost certainly on another workspace. Look for cells in the grid with a medium-brightness fill — those have apps in them. Click through them to find it. You can also right-click the app's taskbar button and choose to move it to the current workspace.
Does Virtual Screen Manager work with games?
Windowed and borderless-windowed games work normally. Exclusive full-screen games (where the game takes full control of the display) cannot be tracked. Running games in borderless-windowed mode is recommended.
What happens to my apps when I close Virtual Screen Manager?
All tracked applications are restored to their normal on-screen positions before Virtual Screen Manager exits. Nothing is closed.
How do I move or resize the grid panel?
Click and drag anywhere on the panel to reposition it. Drag the resize handle in the lower-right corner to resize it. Both are saved automatically.
Settings
Where are my settings stored?
Settings are saved to %AppData%\Virtual Screen Manager\settings.json. Paste that path into File Explorer's address bar to open the folder. Deleting the file resets everything to defaults.
Can I rename workspaces?
Not in this version. Workspace renaming is planned for a future update.
Can I have Virtual Screen Manager start automatically with Windows?
Yes. Check "Start with Windows" in Settings.
How do I uninstall Virtual Screen Manager?
Open Windows Settings > Apps, search for Virtual Screen Manager, and select Uninstall. To fully clean up, you can also delete %AppData%\Virtual Screen Manager manually.
Multi-Monitor
Does Virtual Screen Manager work with multiple monitors?
Yes. Each physical screen gets its own independent workspace grid. You can have up to nine workspaces per screen simultaneously.
Can I sync all monitors to the same workspace?
Yes. Enable "Mirror all monitors" in Settings. When active, switching workspaces on any screen switches all screens together.
"Show secondary monitor panels" is greyed out in Settings. Is something wrong?
No — that option only applies when a second monitor is connected. It will become active as soon as you plug one in.

Does Virtual Screen Manager affect system performance?
No meaningful impact. It uses a lightweight polling timer and low-level Windows calls. CPU and memory usage are negligible.
